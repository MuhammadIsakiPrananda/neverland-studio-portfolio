### Quick Fixes

For typos, broken links, or small improvements:

1. Click "Edit" on the file in GitHub
2. Make your changes
3. Submit a pull request

### Larger Changes

For new documentation or major updates:

1. **Check existing docs** - Avoid duplication
2. **Follow the style** - Match existing format
3. **Test all links** - Ensure they work
4. **Update this guide** - If adding new docs
5. **Submit PR** - With clear description

‚úÖ **Do**:

- Add table of contents for long docs
