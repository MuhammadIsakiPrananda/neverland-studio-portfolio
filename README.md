**Neverland Studio Portfolio** is a comprehensive, production-ready full-stack platform designed for showcasing IT services, managing learning programs, and handling digital consultations. Built with modern technologies and best practices, it provides a robust foundation for educational institutions, IT service providers, and digital agencies.

## ğŸš€ Deployment

### Production Deployment with Cloudflare Tunnel

**Recommended for secure, zero-configuration deployment**

#### Step 1: Install Cloudflare Tunnel

```bash
# Ubuntu/Debian
wget https://github.com/cloudflare/cloudflared/releases/latest/download/cloudflared-linux-amd64.deb
sudo dpkg -i cloudflared-linux-amd64.deb

# Verify
cloudflared --version
```

#### Step 2: Authenticate

```bash
cloudflared tunnel login
```

#### Step 3: Create Tunnel

```bash
cloudflared tunnel create neverland-portfolio
```

#### Step 4: Configure

Create `~/.cloudflared/config.yml`:

```yaml
tunnel: YOUR_TUNNEL_ID
credentials-file: /root/.cloudflared/YOUR_TUNNEL_ID.json

ingress:
  - hostname: yourdomain.com
    service: http://localhost:80
  - service: http_status:404
```

#### Step 5: Deploy Application

```bash
# Start application
docker compose --profile production up -d --build

# Run migrations
docker compose exec backend php artisan migrate --force

# Optimize
docker compose exec backend php artisan optimize
```
